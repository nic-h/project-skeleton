# UI Craft Skill

## Checklist

### Before Coding
- [ ] Read CLAUDE.md design system
- [ ] Check docs/DESIGN_SYSTEM.md
- [ ] Check existing components

### Implementation
- [ ] Uses CSS variables
- [ ] No hardcoded values
- [ ] No inline styles
- [ ] Loading/empty/error states
- [ ] Max 150 lines

### Anti-Patterns
- ❌ Inline styles
- ❌ Hardcoded colors
- ❌ Magic numbers
- ❌ Large files
